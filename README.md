# Power-Bi-Sales-Data:

This project analyzes sales performance data to identify key business insights, such as revenue trends, top-performing salespersons, and product distribution across different regions.
It was developed using Power BI Desktop to visualize KPIs and provide actionable insights for decision-making.

Objectives:

Analyze regional and category-wise sales performance.
Identify the top 5 products and salespersons driving revenue.
Visualize profit margins, discounts, and customer trends.
Enable interactive exploration of key metrics using slicers and filters.

Tools & Technologies:

Power BI Desktop
Excel for data preparation
DAX (Data Analysis Expressions) for calculated measures
Power Query for data transformation

Key Insights:

Identified top 3 performing regions with over 40% of total sales.
Found that 2 product categories contribute 65% of total profit.
Highlighted top-performing sales representatives by revenue.
Visualized monthly trends showing peak sales in Q3.
